#### React Hooks란?React Hooks란?React Hooks란?React Hooks란?

React Hooks는 함수형 컴포넌트에서 상태 관리와 생명주기를 다룰 수 있게 해주는 기능입니다.

#### 주요 Hooks

**useState**: 컴포넌트의 상태를 관리합니다.

```javascript
const [count, setCount] = useState(0);
```

**useEffect**: 컴포넌트의 생명주기와 부수 효과를 처리합니다.

```javascript
useEffect(() => {
  // 컴포넌트 마운트 시 실행
  return () => {
    // 언마운트 시 cleanup
  };
}, [dependencies]);
```

**useMemo**: 계산 비용이 큰 값의 메모이제이션을 통해 성능을 최적화합니다.

이러한 Hooks를 적절히 활용하면 더 깔끔하고 재사용 가능한 컴포넌트를 만들 수 있습니다.
