#### TypeScript 활용하기

TypeScript는 JavaScript에 타입 안정성을 추가한 언어입니다.

#### 유용한 팁들

**1. 타입 추론 활용**: 명시적 타입 선언보다 타입 추론을 활용하세요.

**2. Union Types**: 여러 타입을 허용하려면 union type을 사용하세요.

```typescript
type Status = 'loading' | 'success' | 'error';
```

**3. Optional Chaining**: `?.` 연산자로 안전하게 접근하세요.

**4. 제네릭 활용**: 재사용 가능한 타입을 만들 때 제네릭을 사용하세요.

**5. 타입 가드**: 타입을 좁혀서 안전하게 사용하세요.

TypeScript를 제대로 활용하면 런타임 에러를 줄이고 코드 품질을 높일 수 있습니다.
