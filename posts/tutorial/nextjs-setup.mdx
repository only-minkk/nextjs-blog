#### Next.js란?

Next.js는 React 기반의 풀스택 프레임워크로, 서버 사이드 렌더링과 정적 사이트 생성을 지원합니다.

#### 프로젝트 생성

```bash
npx create-next-app@latest my-blog
```

#### 주요 기능

- **App Router**: 파일 기반 라우팅 시스템
- **Server Components**: 서버에서 렌더링되는 컴포넌트
- **API Routes**: 백엔드 API를 쉽게 만들 수 있습니다

#### 첫 페이지 작성

`app/page.tsx` 파일을 수정하여 원하는 페이지를 만들 수 있습니다.

Next.js를 사용하면 SEO에 최적화된 블로그를 빠르게 만들 수 있습니다.
